## 📌 프로젝트 소개

**블랙홀 이스케이프**에서 사용하는 입력 관리 시스템입니다.

키보드와 게임패드를 모두 지원하며, UI와 게임 플레이 입력을 통합 관리하도록 설계하였습니다.

주요 기능

- ⌨️ 키보드 / 게임패드 입력 관리
- 🎮 게임패드 자동 감지 및 UI 변경
- 🔄 키 리바인딩 지원
- 🖥️ UI 입력 권한 관리
- 📋 메뉴 및 취소 입력 처리
- 📳 게임패드 진동 제어
